# 图书管理系统

## 设计一个图书馆理系统，有用户登陆，包括管理员登录和读者登录
## 管理员实现：新书录入、查询指定书的状态、能查询逾期的书、能查询每个用户借了哪些书，以及借书的历史
## 读者实现：借书、还书、每个用户只能借3本书，两个月内归还、能查询书的状态、能查询借了哪些书，以及借书的历史

## 数据库管理员与读者区别：admin表中status=1表示读者，status=2 表示管理员
